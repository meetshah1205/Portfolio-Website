Software I use

Softwares are a very important thing for programmers. I don't use too many softwares because I like to focus on what code I am writing instead of what medium I am using.

Code Editors/IDEs

NAME

Visual Studio Code

Visual Studio Code (VS Code) is a very popular editor among begginer programmers. I used it a lot as it is versitile and beginner friendly and all the coding tutorials mostly use it. But now I am slowly shifting to Sublime Text.

NAME

Sublime Text

Sublime text was my choice as my secondary editor as I thought the interface looked minimal and it did not force you into installing 20 extentions just to write good code. Sublime text and VS code combined were used to make this website 😁.

NAME

JetBrains Fleet

I haven't used Fleet as much as VS Code and Sublime, but my experience using it was good. JetBrains is ofcourse known for making heavy and powerful IDEs, but Fleet as an exception being lightweight yet very powerful is very good.

And that's it. That's like all the software I use.
